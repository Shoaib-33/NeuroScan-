Dynamic Quantization converts model weights to INT8 at export time. Activations remain FP32 at runtime and are quantized on-the-fly. No calibration data needed.
Static Quantization quantizes both weights and activations to INT8. Requires calibration data. It can be smaller and faster, but accuracy may vary by architecture.
